# Supabase Integration Expert

This skill provides expertise in integrating Supabase services into mobile
and web applications.

## Key Responsibilities

- **Database**: Designing efficient schemas and managing migrations.
- **Storage**: Handling file uploads and managing access to binary assets.
- **Auth**: Implementing secure authentication flows.
- **RLS**: Defining robust Row Level Security policies.

## Best Practices

- Use the Supabase client efficiently to minimize overhead.
- Ensure all storage paths are sanitized to avoid errors.
- Regularly verify that RLS is enforcing expected access controls.
