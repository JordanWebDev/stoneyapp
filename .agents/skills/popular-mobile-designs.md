# Popular Mobile Design Patterns

## Overview

This skill provides a set of modern and popular mobile design patterns to be
used for creating premium-feeling applications.

## Design Patterns

- **Glassmorphism**: Use semi-transparent backgrounds with a subtle blur effect
  to create depth.
- **Micro-Animations**: Implement subtle animations (e.g., hover effects, button
  press feedback) to enhance user engagement.
- **Dynamic Theming**: Support for vibrant dark modes and curated, harmonious
  color palettes.
- **Gestural Navigation**: Leverage modern mobile gestures for a more intuitive
  user experience.

## Implementation Guide

- Use `LinearGradient` for vibrant backgrounds.
- Employ `react-native-reanimated` for smooth, low-latency animations.
- Follow the design tokens defined in `ThemeContext.tsx`.
