# Stoney Nakoda: Language Journey

A modern, premium language learning application for the Stoney Nakoda language.
Built with React Native and Expo, featuring a Supabase backend.

## Features

- **Learn**: Browse vocabulary by category.
- **Quiz**: Test your knowledge with interactive quizzes.
- **Flashcards**: Standard flashcard learning mode.
- **Match**: A fun matching game for words and translations.
- **Wordle & Crossword**: Daily word games to keep learning engaging.
- **Admin Dashboard**: Effortlessly manage vocabulary and view user feedback.

## Tech Stack

- **Frontend**: React Native, Expo, Expo Router.
- **Backend**: Supabase (Database & Storage).
- **Styling**: Vanilla CSS-in-JS with a premium zinc and orange theme.

## Development

1. Install dependencies: `npm install`.
2. Start the development server: `npm run dev`.
3. Open on iOS, Android, or Web.
